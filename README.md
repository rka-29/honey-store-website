# Honey Store Website

A full-stack e-commerce website for a honey store built using HTML, CSS, JavaScript, Firebase, and Firebase Hosting.

## Features

- Product listings
- Shopping cart functionality
- Order handling
- Admin page
- Firebase backend integration
- Live deployment using Firebase Hosting

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase
- Firebase Hosting
